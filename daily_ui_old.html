Mobile App Concepts

UI Design

Overview

In an effort to up level my user interface design skills and increase the number of mobile app UI projects in my portfolio, I signed up for the Daily UI Challenge, a 100-day challenge sponsored by dailyui.co. Every morning I wake up with a new UI design problem to solve in my inbox in whatever way I see fit. The prompts are purposefully brief and open-ended, allowing for maximum creativity, which makes them all the more interesting and fun to complete. I have chosen to complete each in mobile size because I enjoy designing for mobile most — I like the extra challenge the minimized screen real estate provides and I like to think doing so stretches my abilities and skills. I have chosen a selection of my favorite designs to share below. I have included the prompt with each.

Technologies Used

Designs

House Monitoring Dashboard

Design a home monitoring dashboard. Be creative! What would make a dashboard visually appealing and fun to use, while also being mindful of the data?

Calculator

Design a calculator. Standard, scientific, or specialty calculator for something such as a mortgage? Is it for a phone, a tablet, a web app?

Credit Card Checkout

Design a credit card checkout form or page. Don't forget the important elements such as the numbers, dates, security numbers, etc.

User Profile

Design a user profile and be mindful of the most important data, names, imagery, placement, etc. Is it for a serious profile? A social profile?

Settings Dashboard

Design settings for something. Is it for security or privacy settings? Game settings? What is it and what's important?

Landing Page

What's the main focus? Is it for a book, an album, a mobile app, a product? Consider important landing page elements (call-to-actions, clarity, etc.).

404 Page

Design a 404 page. Does it suit the brand's style? Is it user-friendly?

Music Player

Design a music player. Consider the controls, placements, imagery such as the artist or album cover, etc.
